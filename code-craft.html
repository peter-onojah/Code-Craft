About Code-Craft

Code Craft is is an intensive 30 day coding exercise which focuses on equiping participants with the requisite skills required in modern technology demands. The Course covers areas such as web design and development as well as programming with Python. The course is held virtually everyday and participants are given projects to work on daily. It is expected that at the end of the course, participants should have garnered enough of the basic knowledge to further into the tech space as programmers or web developers.

About the Host

Sir Innocent Jeremiah is a Computer Science undergraduate of the Benue State university, Makurdi. He is a dedicated Tutor and tech enthusiast, he focuses on empowering the teeming unemployed Nigerian Youth with skills in software engineering. He palns to acheive this through the provision of comprehensive, accessable and affordable software development courses notably amoungst which are INNOVATE-A-THON and CODE-CRAFT.
